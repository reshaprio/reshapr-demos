# MCP Apps

This directory contains MCP app examples used by the reShapr demos.

Apps are managed as npm workspaces from the repository root. Install once:

```sh
npm install
```

Start one app at a time:

```sh
npm run start:open-meteo
npm run start:github-user
npm run start:github-user-prompts
```

Build all apps:

```sh
npm run build
```
