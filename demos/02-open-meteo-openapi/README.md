# Demo 02: Open-Meteo OpenAPI

After this demo imports the Open-Meteo service, run Demo 07 to attach the MCP app resource.

Run from this directory:

```sh
./demo.sh
```

Run from the repository root:

```sh
./demos/02-open-meteo-openapi/demo.sh
```

Or use the npm alias from the repository root:

```sh
npm run demo:02
```
